# Study 项目说明

本项目是一个前端学习演示项目，包含多种前端效果实现和练习。


## 主要功能

1. 图片花式展示 - 使用Three.js实现的3D图片展示效果
2. 爱情故事专题 - 带有音乐和图片展示的浪漫主题页面
3. 音频播放功能 - 多个页面集成了音频播放功能
4. 响应式设计 - 适配不同设备的布局

## 使用说明

1. 直接打开index.html即可查看主页面
2. 各子目录中的HTML文件可独立运行
3. 需要现代浏览器支持(Chrome/Firefox/Edge最新版)

## 技术栈

- HTML5
- CSS3
- JavaScript
- Three.js
- jQuery
